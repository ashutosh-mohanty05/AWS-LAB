# AWS Cloud Hands-On Labs

This repository contains practical AWS projects for learning real-world cloud infrastructure.

## Labs Included

1. VPC Networking
2. EC2 & Load Balancer
3. Auto Scaling
4. RDS Database
5. S3 Static Website Hosting
6. CloudFront CDN
7. IAM Security
8. CloudWatch Monitoring
9. Terraform Infrastructure

## Goal

To practice production-style AWS architectures and strengthen cloud engineering skills.
